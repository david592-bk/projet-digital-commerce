# Kindu Frontend

Frontend React pour la plateforme Kindu Commerce Digital.

## Installation

```bash
cd "commerce_digital_kindu/frontend"
npm install
```

## Lancement

```bash
npm run dev
```

## Remarques

- Le frontend charge les données via l’API Django exposée sur `http://127.0.0.1:8000/api`.
- Le design est responsive grâce à Tailwind CSS.
